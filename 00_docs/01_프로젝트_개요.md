# 프로젝트 개요

> **독서 취향 분석 플랫폼** - AI 기반 개인 독서 관리 및 취향 분석 SNS

---

## 🎯 프로젝트 목표

### 개인 목표
1. **기술 습득**: Next.js 14, React Query, OpenAI API 등 최신 기술 스택 학습
2. **포트폴리오**: 차별화된 개인 프로젝트로 이직 준비
3. **완성**: 4-5개월 내 MVP 배포 및 단계적 기능 확장

### 비즈니스 목표
- **문제 정의**: 독서 기록은 많지만, 자신의 독서 취향을 파악하기 어려움
- **솔루션**: AI 분석으로 독서 성향을 파악하고, 비슷한 취향의 독서가와 연결
- **타겟**: 20-40대 독서 애호가, MZ세대 자기계발 관심층

---

## 💡 핵심 차별화 포인트

| 구분 | 기존 서비스 | 본 플랫폼 |
|-----|-----------|----------|
| **포커스** | 책 추천에 집중 | **취향 분석**에 집중 |
| **리뷰** | 단순 별점/리뷰 | **AI 기반 성향 분석** |
| **추천** | 일방적 추천 | **비슷한 취향 매칭** |
| **데이터** | 정량적 통계 | **감성적 인사이트** |

### 밀리의 서재 벤치마크
- ✅ **무제한 구독** → 우리는 **무료 + AI 제한** (프리미엄 모델)
- ✅ **큐레이션** → 우리는 **AI + 사용자 매칭**
- ✅ **커뮤니티** → 우리는 **취향 기반 연결**

## 📅 개발 로드맵

### Phase 1: MVP (Week 1-4) 🥇
**목표**: 혼자 사용 가능한 독서 다이어리

```
필수 기능:
├─ Google 소셜 로그인
├─ 알라딘 API 책 검색
├─ 독서 기록 CRUD (읽고 싶은 책/읽는 중/완독)
├─ 개인 책장 (필터링, 정렬)
└─ 기본 프로필 (독서 통계)
```

**예상 시간**: 4주
**마일스톤**: Vercel 배포 + 본인이 실사용

---

### Phase 2: 소셜 네트워크 (Week 5-8) 🥈
**목표**: SNS로 전환, 네트워크 효과 시작

```
소셜 기능:
├─ 팔로우/팔로워 시스템
├─ 피드 (타임라인)
├─ 댓글 & 좋아요
├─ 기본 알림
└─ 책 공개/비공개 설정
```

**예상 시간**: 4주
**마일스톤**: 친구 초대 & 베타 테스트

---

### Phase 3: AI 인텔리전스 (Week 9-12) 🥉
**목표**: 핵심 차별화 기능 완성

```
AI 기능:
├─ OpenAI 취향 분석 (성향, 선호 장르, 패턴)
├─ 취향 비교 (유사도 %, 공통 관심사)
├─ AI 책 추천 (이유 설명 포함)
└─ 추천 알고리즘 개선
```

**예상 시간**: 4주
**마일스톤**: AI 분석 정확도 80%+

---

### Phase 4: 고급 기능 (Week 13-16) 🏆
**목표**: 장기 사용자 유지

```
게이미피케이션 & 데이터:
├─ 독서 통계 & 차트 (월별, 장르별)
├─ 독서 캘린더 (GitHub 스타일 히트맵)
├─ 독서 챌린지 (목표 설정, 진행률)
├─ 랭킹 시스템 (독서왕, 리뷰어)
└─ 뱃지/업적 시스템
```

**예상 시간**: 4주
**마일스톤**: MAU(월간 활성 사용자) 증가

---

### Phase 5: 폴리싱 (Week 17-20) ✨
**목표**: 완성도 향상 및 런칭 준비

```
최종 마무리:
├─ 배너/이벤트 시스템
├─ 테마별 큐레이션
├─ UI/UX 개선 (애니메이션, 마이크로 인터랙션)
├─ 성능 최적화 (로딩 속도, SEO)
└─ 반응형 디자인 완성
```

**예상 시간**: 4주
**마일스톤**: 정식 런칭 + 마케팅

## 🚫 하지 않을 것 (명확한 범위 설정)

### 제외 기능
- ❌ 전자책 뷰어 (밀리의 서재와 경쟁하지 않음)
- ❌ 책 구매 기능 (제휴 링크만)
- ❌ 실시간 채팅 (댓글로 충분)
- ❌ 복잡한 알림 (이메일 알림 없음)

### 제외 이유
- **집중**: 취향 분석이라는 핵심 가치에 집중
- **범위**: 4-5개월 안에 완성 가능한 범위로 제한
- **차별화**: 모든 기능을 따라하지 않고, 우리만의 강점 살리기

## 🎯 경쟁 우위

### vs 독서 기록 앱
- ✅ **AI 분석**으로 인사이트 제공
- ✅ **소셜 기능**으로 네트워크 효과

### vs 독서 SNS
- ✅ **취향 매칭**으로 관련성 높은 연결
- ✅ **데이터 시각화**로 동기부여

### vs 밀리의 서재
- ✅ **무료 + 가벼움** (구독 부담 없음)
- ✅ **커뮤니티 중심** (큐레이션보다 사용자 중심)
