```
    └─ following 수 (Phase 3 이후)

[ ] 월별 독서량 차트
    ├─ Bar Chart (1월~12월)
    ├─ 권수 기준

[ ] 읽은 책 목록 (하단)
    ├─ 책 표지 그리드
    ├─ 카테고리별 필터 탭

```
