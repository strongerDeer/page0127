# 핵심 기능 명세

> 개발 시 참고할 주요 기능 목록 및 우선순위

---

## Phase 1: MVP (Week 1-4) 🚀

### 1. 사용자 인증
```
[ ] Google OAuth 로그인
[ ] 프로필 설정
    ├─ 닉네임
    ├─ 소개
    ├─ 프로필 이미지
    └─ 연간 독서 목표 (선택)
[ ] 로그아웃
[ ] 계정 삭제
```

### 2. 책 검색 및 등록
```
[ ] 알라딘 API 연동
    ├─ 실시간 검색 (디바운싱)
    ├─ 검색 결과 페이지네이션
    └─ 책 상세 정보 표시
[ ] 내 서재에 추가
    ├─ 독서 상태 선택 (읽고 싶은/읽는 중/완독)
    ├─ 별점 (1-5점)
    └─ 한줄평 (선택)
```

### 3. 독서 기록 관리
```
[ ] CRUD 기능
    ├─ CREATE: 책 추가
    ├─ READ: 책 목록 조회
    ├─ UPDATE: 정보 수정
    └─ DELETE: 책 삭제
[ ] 독서 상태 변경
    ├─ 읽고 싶은 책
    ├─ 읽는 중
    └─ 완독
[ ] 읽은 날짜 기록 (선택)
[ ] 공개/비공개 설정
```

### 4. 개인 책장
```
[ ] 상태별 탭
    ├─ 읽고 싶은 책
    ├─ 읽는 중
    └─ 완독
[ ] 필터링
    ├─ 장르별
    ├─ 별점별
    └─ 연도별
[ ] 정렬
    ├─ 최신순
    ├─ 오래된순
    ├─ 별점 높은순
    └─ 제목순
[ ] 검색 (내 책장 내)
[ ] 그리드/리스트 뷰 전환
```

### 5. 기본 프로필 페이지
```
[ ] 독서 통계
    ├─ 총 읽은 책
    ├─ 올해 읽은 책
    ├─ 평균 별점
    └─ 좋아하는 장르 TOP 3
[ ] 최근 읽은 책 (5권)
[ ] 프로필 편집 버튼
```

## Phase 3: AI 인텔리전스 (Week 9-12) 🤖

### 10. OpenAI 취향 분석
```
[ ] 독서 성향 분석
    ├─ 성향 타입 (예: "감성적 독서가")
    ├─ 상세 설명 (200자)
    └─ 독서 패턴 (예: "주말 집중형")
[ ] 선호 장르 분석
    ├─ 장르별 점수 (0-100)
    ├─ TOP 3 장르
    └─ 장르별 읽은 권수
[ ] 키워드 추출
[ ] 월 3회 무료 제한
```

### 11. 취향 비교 기능
```
[ ] 다른 사용자와 비교
    ├─ 유사도 (%)
    ├─ 공통 관심 도서
    ├─ 공통 장르
    └─ 취향 차이 시각화
[ ] AI 코멘트 (예: "두 분 모두 미스터리를 좋아하시네요!")
[ ] 비교 결과 공유 (선택)
```

### 12. AI 책 추천
```
[ ] 내 취향 기반 추천
    ├─ 추천 도서 3-5권
    ├─ 추천 이유 설명
    └─ 유사 도서 찾기
[ ] 추천 피드백 (좋아요/별로)
```

## Phase 4: 고급 기능 (Week 13-16) 📊

### 13. 독서 통계 & 시각화
```
[ ] 대시보드
    ├─ 월별 독서량 (막대 차트)
    ├─ 장르 분포 (파이 차트)
    ├─ 독서 추세 (라인 차트)
    └─ 연간 요약
[ ] 독서 스트릭
    ├─ 연속 읽기 일수
    └─ 최장 기록
[ ] 비교 통계
    ├─ 전년 대비
    └─ 평균 대비
```

### 14. 독서 캘린더
```
[ ] GitHub 스타일 히트맵
[ ] 날짜별 독서 기록
[ ] 월간/연간 뷰 전환
[ ] 날짜 클릭 → 해당 날짜 책 표시
```

### 15. 독서 챌린지
```
[ ] 챌린지 종류
    ├─ 권수 목표 (예: "2025년 50권 읽기")
    ├─ 장르 다양성 (예: "5가지 장르 도전")
    └─ 연속 읽기 (예: "30일 연속 독서")
[ ] 진행률 표시
[ ] 완료 시 뱃지 획득
[ ] 커뮤니티 챌린지 참여 (선택)
```

### 16. 랭킹 시스템
```
[ ] 리더보드
    ├─ 이번 달 독서왕 (권수)
    ├─ 열정적인 리뷰어 (리뷰 수)
    └─ 장르별 전문가
[ ] 주간/월간/연간 탭
[ ] 내 순위 표시
```

### 17. 뱃지/업적 시스템
```
[ ] 뱃지 종류
    ├─ 🏅 첫 걸음: 첫 책 등록
    ├─ 📚 열정적인 독서가: 10권 완독
    ├─ 🔥 불타는 열정: 30일 연속
    ├─ 🌟 다양성의 달인: 10개 장르
    └─ 👑 리뷰 마스터: 50개 리뷰
[ ] 뱃지 목록
[ ] 획득 조건 표시
[ ] 프로필에 표시
```
