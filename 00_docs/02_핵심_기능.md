### 5. 독서 통계 대시보드 ⭐ (MVP 핵심) - "내 서재" 통합

**개념**: 대시보드 = 내 서재 (통계 + 책 목록 통합)

```
[x] 전체 독서 통계 (All Time Stats) - 신규
    ├─ 🏆 독서 여정 카드
    │   ├─ 총 읽은 책 (권) + 10점 만점 책 권수/비율
    │   ├─ 총 읽은 쪽수 + 하루 평균 쪽수
    │   ├─ 독서 기간 (시작일 ~ 현재)
    │   └─ 예상 독서 시간 (시간/일 환산)
    ├─ 📊 카테고리별 분포 (Pie Chart)
    │   ├─ Top 5 카테고리 + 기타
    │   ├─ 비율 + 권수 함께 표시
    │   └─ 클릭 시 해당 카테고리 필터 적용 (예정)
    ├─ 📈 최근 5년 독서량 (Bar Chart)
    │   ├─ 연도별 권수 트렌드
    │   ├─ 전년 대비 증가율 표시
    │   └─ 최고 기록 연도 하이라이트
    └─ ⭐ 평점 분포 (Horizontal Bar)
        ├─ 0~10점 분포 (권수 + 비율)
        ├─ 평균 평점 표시
        └─ AI 인사이트 메시지 ("엄격한 평가자" 등)

[x] 연도별 통계 (선택된 연도 기준)
    ├─ 연도 선택 드롭다운
    ├─ 해당 연도 읽은 책/쪽수/평균 평점
    └─ 연간 목표 진행률 프로그레스 바 (예정)
[x] 월별 독서량 차트 (Bar Chart)
    ├─ 1월~12월 전체 표시

[x] 카테고리별 레이더 차트
```

## Phase 2: AI 취향 분석 (Week 7-10) 🤖

**목표**: AI 기반 독서 성향 분석 및 맞춤 추천

**사용 범위**: 관리자 전용 (월 3만원 예산 내 운영)

### 7. 내 취향 분석 (대시보드)

**실행 방식**: 대시보드에서 "내 취향 분석하기" 버튼 클릭 (수동)

**분석 데이터**:
- 사용자의 완독한 책 목록
- 책별 별점 (0-10점)
- 알라딘 API 책 정보 (제목, 저자, 카테고리, 책 소개, 목차)
- 독서 기간 (시작일~종료일)

**AI 분석 항목**:

```
[x] 1. 당신의 독서 성향
    ├─ 책 소개글 텍스트 분석 (높은 점수 vs 낮은 점수 책)
    ├─ 목차 구조 패턴 분석
    ├─ 숨겨진 주제 연결고리 발견
    └─ 성향 타입 분류 (예: "내면 탐구형", "질문 즐기는 독자")

[x] 2. 당신의 선호도 프로필
    ├─ 좋아하는 주제/스타일/구조 (책 소개 + 목차 기반)
    ├─ 피하는 주제/스타일/구조
    ├─ 페이지 수/저자 성향/출판 시기 선호도
    └─ 카테고리 조합 분석 (예: "에세이 + 심리학" 선호)

[x] 3. 맞춤 추천 도서 (각 5권, 국내도서 한정)
    ├─ A. 성향 맞춤: 기존 취향에 딱 맞는 안전한 추천
    ├─ B. 성향 확장: 비슷하지만 조금 다른 영역으로 확장
    └─ C. 챌린지: 안 읽었지만 좋아할 가능성 있는 새로운 영역

[x] 분석 결과 저장
    ├─ 분석 일시 기록
    ├─ 결과 캐싱 (재분석 최소화)
    └─ 책 5권 이상 추가 시 재분석 권장

[x] 백엔드 구현 완료 (2025-12-21)
    ├─ DB 스키마 설계 및 마이그레이션
    ├─ OpenAI API 연동 (GPT-4o-mini)
    ├─ API 엔드포인트 (POST /api/taste-analysis/analyze)
    ├─ 프롬프트 템플릿 작성
    └─ TypeScript 타입 정의

[ ] UI 구현 (남은 작업)
    ├─ 대시보드에 "내 취향 분석하기" 버튼
    ├─ 분석 결과 표시 페이지 (/dashboard/taste-analysis)
    └─ 로딩 상태 및 에러 처리
```

**AI API 선택**:
- ✅ OpenAI GPT-4o-mini (선택됨 - 비용 효율적)
- 예상 비용: 1회 분석 약 1원 (실제 측정 기준)

### 8. 독서 궁합 측정 (타인 책장)

**실행 방식**: 다른 사람 공개 책장에서 "독서 궁합 측정하기" 버튼 클릭

**비교 데이터**:
- 나와 상대방의 완독한 책 목록
- 각자의 별점 데이터
- 책 메타데이터 (카테고리, 책 소개 등)

**AI 분석 항목**:

```
[ ] 1. 궁합 점수
    ├─ 전체 궁합도 (0-100%)
    ├─ 계산 근거 설명
    └─ 궁합 타입 분류 (예: "서로 다른 세계", "영혼의 책벗")

[ ] 2. 독서 패턴 및 관심 장르 유사도
    ├─ 공통 관심 장르/주제 (책 소개 텍스트 분석)
    ├─ 독서 성향 비교 (예: 나는 "깊이형", 상대는 "다독형")
    ├─ 별점 패턴 유사도 (엄격한 평가자 vs 후한 평가자)
    └─ 차이점 & 공통점 인사이트

[ ] 3. 서로에게 추천할 수 있는 책 (각 3권씩, 국내도서)
    ├─ 내가 읽었지만 상대는 안 읽은 책 중 추천
    ├─ 상대가 읽었지만 내가 안 읽은 책 중 추천
    ├─ 추천 이유 (궁합 분석 기반)
    └─ 함께 읽으면 좋을 책 (토론 포인트 제시)

[ ] 궁합 결과 저장
    ├─ 측정 일시 기록
    ├─ 결과 캐싱
    └─ 책장 변경 시 재측정 권장
```

**비용 관리 전략**:
```
✅ 캐싱: 한 번 분석한 결과 DB 저장
✅ 배치 처리: 여러 책 정보 한 번에 분석
✅ 저렴한 모델: GPT-4o-mini 또는 Claude Haiku
✅ 분석 주기: 신규 책 5권 추가 시 또는 수동 재분석
→ 월 3만원 예산으로 충분히 운영 가능
```

## 📝 Phase 1 완료 현황 (2025-12-10)

### ✅ 완료된 주요 기능

**독서 통계 대시보드 (MVP 핵심 완성)**

*전체 독서 통계 (All Time Stats)*
- 독서 여정 카드: 총 읽은 책, 10점 만점 비율, 총 쪽수, 하루 평균, 독서 기간, 예상 독서 시간
- 카테고리 분포 파이 차트: Top 5 카테고리 + 기타, 비율/권수 표시
- 5년 독서량 트렌드: 연도별 Bar Chart, 전년 대비 증가율, 최고 기록 하이라이트
- 평점 분포: Horizontal Bar, AI 인사이트 메시지

*연도별 통계 (선택된 연도)*
- 연도별 필터링: URL 쿼리 파라미터 기반, 자동 연도 목록 생성
- 3개 차트 시각화: 월별 Bar Chart, 카테고리 Radar Chart, 평점 Doughnut Chart
- 복합 필터링 시스템: 상태 + 월 + 카테고리 + 평점 (4가지 필터 조합)
- 페이지네이션: 12권/페이지, 필터 변경 시 첫 페이지 자동 이동
- 평점 시스템: 0, 1, 2, 3, 4, 5, 10점 (7가지) 지원

*내 책장 (책 목록)*
- 상태별 탭: 전체/완독/읽는 중/읽고 싶은 (4개 탭)
- 정렬 기능: 최신순/오래된순/별점 높은순/낮은순/제목순 (6가지)
- 활성 필터 뱃지: 클릭으로 개별 필터 제거
- /books 페이지 리다이렉트: 대시보드로 통합

**기술 스택**
- Recharts: 차트 라이브러리
- Server/Client Component 분리: 데이터 페칭 최적화
- URL 상태 관리: 연도 필터 지속성
- Lift State Up 패턴: 필터 상태 중앙 관리

*연간 독서 목표*
- 목표 설정 다이얼로그: 연도 선택, 목표 권수 입력 (1~1,000권)
- 진행률 카드: 프로그레스 바, 달성률, 남은 권수 표시
- 자동 계산: 선택된 연도의 완독 권수 실시간 반영
- 프로필 시스템: profiles 테이블, reading_goal JSONB 컬럼

**기술 구현**
- Profile Entity: 타입 정의, API (getProfile, upsertProfile, updateReadingGoal)
- UI 컴포넌트: ReadingGoalDialog, ReadingGoalProgress
- Shadcn 컴포넌트: Label, Progress
- 자동 프로필 생성: 첫 로그인 시 자동 upsert
