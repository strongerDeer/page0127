# NextJS

```bash
npx create-next-app@latest . --typescript --tailwind --app --no-src-dir
```

- `.`: 현재 폴더에 설치
- `--typescript`: TypeScript 사용 (타입 안정성)
- `--tailwind`: Tailwind CSS 자동 설정 (유틸리티 CSS)
- `--app`: App Router 사용 (Next.js 14의 최신 라우팅 방식)
- `--no-src-dir`: src 폴더 없이 루트에 app 폴더 생성
